# ecommerce-admin

🛍又一套从零开始开发的电商管理系统，欢迎参与开发或者提交 issue 讨论

## 项目启动

### 安装依赖

```bash
yarn install
```

或者

```bash
npm install
```

### 开发环境启动

> 同时启动了前端项目和数据 mock 服务，mock 服务在 3000 端口

```bash
yarn start
```

或者

```
npm start
```

### 生产环境打包

```bash
yarn build
```

或者

```bash
npm run build
```

### 运行单元测试

```bash
yarn test:unit
```

或者

```bash
npm run test:unit
```

### 代码格式检查

```bash
yarn lint
```

或者

```bash
npm run lint
```
